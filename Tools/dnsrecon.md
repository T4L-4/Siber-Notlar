Standart kullanım:

    dnsrecon -d example.com 

Wordlist ile kullanım:

    dnsrecon -d example.com -D /usr/share/wordlists/dnsmap.txt 

XML çıktı alma:

    dnsrecon -d example.com -D /usr/share/wordlists/dnsmap.txt -t std --xml dnsrecon.xml
